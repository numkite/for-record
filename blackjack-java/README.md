# BlackJack-Java

딜러와 플레이어가 함께 카드의 숫자를 겨루는 것으로 2장 이상의 카드를 뽑아 그 합계를 21점에 가깝도록 만들어 승부하는 카드게임


## Project Structure
- Blackjack.jar 실행 시 즉시 게임이 실행됩니다.
- src 폴더에는 프로젝트에 사용된 모든 Java 소스 코드가 포함되어 있습니다.

```
blackjack-java/
│
├── project_guidelines/
│   └── project.pdf
│
├── src/                          # 블랙잭 게임 전체 Java 소스 코드 (.java 파일)
│   └── ...                       # 카드, 덱, 플레이어, 게임 로직 등 클래스들
│
├── Blackjack.jar
├── user_data.txt
├── 블랙잭 프로젝트 보고서.pdf
└── README.md
```

## Rules
1. 처음 회원가입한 모든 회원들에게 칩 10개를 부여한다.
2. 자신이 가지고 있는 칩 갯수 이내에서 1개 이상의 칩을 배팅한다.
3. 시작하고 딜러와 플레이어는 각각 2장의 카드를 받는다. 그 중 딜러의 카드는 한 장만 공개된다.
4. 플레이어는 카드의 합이 21에 도달하기 전까지 카드를 더 뽑을지 선택할 수 있다.
5. 플레이어의 카드 뽑기가 끝나면 딜러의 카드 뽑기가 시작된다. 
6. 딜러는 카드 합가 16을 넘을 때까지 카드를 뽑는다.
7. 플레이어와 플레이어의 카드 합을 비교해서 21에 가장 가까운 이가 승리한다.
8. 합이 21이 된다면 Blackjack으로 플레이어는 배팅한 칩 갯수의 2배를 얻는다.

   -단, 플레이어와 딜러 모두  합이 21이라면 플레이어가 블랙잭으로 승리한다.
10. 21 미만이지만 딜러보다 카드 합이 높다면 배팅한 만큼의 칩을 얻는다.
11. 카드의 합이 21을 넘는다면 배팅한 만큼 칩을 잃는다.
12. 플레이어와 딜러의 점수가 같다면 비긴 것으로 배팅한 칩은 변경되지 않는다.
